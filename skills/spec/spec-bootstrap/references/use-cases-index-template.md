# Use Cases Index Template

## Usage
Index file for the `specs/use-cases/` folder.
One section per module. Links to the module's own index and to each use case directly.

---

```markdown
---
type: use-cases-index
module: [domain-name]
---

# Use Cases

## [ModuleName]
→ [Module index](./[module-name]/index.md)

- [[UseCaseName]]([module-name]/[use-case-name].md)
```
