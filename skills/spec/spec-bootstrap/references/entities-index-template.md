# Entities Index Template

## Usage
Index file for the `specs/entities/` folder.
One row per entity. Status pulled from each entity file's frontmatter.

---

```markdown
---
type: entities-index
module: [domain-name]
---

# Entities

| Entity | Module | Status |
|--------|--------|--------|
| [[EntityName]]([entity-name].md) | [module] | draft |
```
