# Drafts & Raw Notes

This folder contains **unprocessed, rough notes** that haven't been formalized yet:

- 📝 Meeting notes
- 💡 Feature ideas
- 🐛 Issues & bugs to fix
- ✅ TODO lists
- 📋 Client requests
- 🗂️ Random research

**Purpose**: Drop everything here first. Don't worry about organization or formatting.

**Workflow**:
1. **Capture** - Write notes as you go (meetings, ideas, bugs)
2. **Review** - Periodically go through drafts folder
3. **Promote** - Move formalized specs to [../entities/](../entities/) or [../usecases/](../usecases/)
4. **Archive or Delete** - Clean up old notes

## Rules

✅ **Do**:
- Write quick, rough notes
- Use bullet points, not prose
- Date your notes
- Link to code/files when relevant
- Keep it messy - this is a scratch space

❌ **Don't**:
- Spend time formatting
- Worry about being complete
- Let notes sit here forever (review regularly)

---

**When a draft becomes clear**, move it:
- Entity spec → `../entities/EntityName.md`
- Use case → `../usecases/UseCaseName.md`
- Update the respective index files
