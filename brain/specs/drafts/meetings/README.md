# Meeting Notes

Raw notes from client meetings, team discussions, and planning sessions.

**Format**: Keep it simple - date, attendees, notes. Don't overthink it.

---

## Template

```markdown
# Meeting - YYYY-MM-DD - Topic

**Attendees**: Names  
**Type**: Client Meeting / Team Planning / Demo

## Agenda
- Topic 1
- Topic 2

## Notes
- Raw notes here
- Bullet points
- Decisions made
- Action items

## Action Items
- [ ] Task 1 - @person
- [ ] Task 2 - @person

## Follow-up
Next meeting date, things to prepare
```
